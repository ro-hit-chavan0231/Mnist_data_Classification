# Simple MNIST Image Classifier

This project demonstrates a basic image classification task using a Neural Network built with Keras (TensorFlow) on the popular MNIST dataset. The goal is to train a model that can accurately identify handwritten digits (0-9).

## Key Features:

*   **Data Loading & Preprocessing:** Efficient loading of the MNIST dataset and normalization of pixel values.
*   **Neural Network Architecture:** Implementation of a simple feed-forward neural network with `Dense` layers.
*   **Model Training:** Training the classifier using `SparseCategoricalCrossentropy` loss and `SGD` optimizer.
*   **Performance Evaluation:** Visualizing training history (accuracy and loss) and evaluating the model's performance on unseen test data.
*   **Prediction Visualization:** Displaying individual predictions to visually inspect the model's accuracy.

This project serves as an excellent starting point for understanding fundamental concepts in deep learning and image classification.
